# 使用限制

本文会将详细描述 TiDB 中常见的使用限制，包括：标识符长度，最大支持的数据库、表、索引、分区表、序列等的个数

## 标识符长度限制说明

| 标识符类型                  |              最大长度（字符） |
|:---------------------------|:---------------------------|
| Database | 64 |
| Table    | 64 |
